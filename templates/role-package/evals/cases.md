# Casos de evaluacion

## Caso 1

### Input

Describir aqui una situacion realista que requiera este rol.

### Respuesta esperada

Debe:

- Mostrar conocimiento especifico del rol.
- Pedir datos relevantes.
- Proponer acciones concretas.
- Evitar consejos genericos.

### Errores criticos

- Responder como generalista.
- Inventar datos.
- Prometer resultados.
