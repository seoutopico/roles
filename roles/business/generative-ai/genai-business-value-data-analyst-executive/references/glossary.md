# Glosario

Terminos que reducen ambiguedad al analizar proyectos de IA generativa para negocio.

| Termino | Definicion operativa | Errores comunes |
| --- | --- | --- |
| Baseline | Medida del estado actual antes de IA: coste, tiempo, calidad, error, conversion, volumen, satisfaccion o riesgo. | Medir solo despues del piloto y llamar impacto a cualquier mejora percibida. |
| Outcome de negocio | Resultado que importa fuera del equipo tecnico: margen, ingresos, retencion, riesgo, velocidad, calidad, experiencia o capacidad operativa. | Confundir entregable, demo o uso con outcome. |
| Metrica operativa | Indicador del proceso afectado: tiempo de ciclo, tickets resueltos, errores, throughput, rework, backlog o coste por tarea. | Usar solo metricas de modelo sin conectar con operacion. |
| Metrica de adopcion | Evidencia de uso real: usuarios activos, frecuencia, recurrencia, cobertura del workflow, abandono, excepciones y satisfaccion. | Contar licencias, accesos o formaciones como adopcion efectiva. |
| Evals | Pruebas sistematicas que describen comportamiento esperado, datos de test y criterios de aceptacion del sistema GenAI. | Evaluar con ejemplos anecdoticos o solo con opinion de stakeholders. |
| Agente | Sistema que puede planificar o ejecutar pasos usando herramientas, memoria o entorno, con algun grado de autonomia. | Llamar agente a cualquier chatbot. |
| RAG | Patron que combina recuperacion de informacion con generacion para responder usando contexto documental o datos externos. | Suponer que RAG elimina alucinaciones sin evaluacion ni control de fuentes. |
| Coste por tarea | Coste completo de ejecutar una tarea con IA: modelo, integracion, revision humana, errores, soporte, monitoreo y mantenimiento. | Mirar solo precio por token o licencia. |
| Ahorro capturable | Parte del ahorro teorico que realmente puede convertirse en menos coste, mas capacidad, mas ingresos o menor riesgo. | Declarar ahorro por minutos liberados sin decidir que se hara con esa capacidad. |
| Hipotesis de valor | Supuesto verificable que conecta una capacidad GenAI con un resultado de negocio. | Presentar aspiracion estrategica como valor probado. |
| Owner operativo | Persona o area responsable de que el sistema se use, mida, mejore y gobierne despues del piloto. | Dejar ownership en el equipo de innovacion una vez terminada la demo. |
| Narrativa ejecutiva | Sintesis orientada a decision: problema, evidencia, implicacion, recomendacion y condicion de exito. | Convertir la presentacion en explicacion tecnica o marketing interno. |
